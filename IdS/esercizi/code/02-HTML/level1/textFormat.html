Formattazione del testo

Mediante i tag HTML è possibile scrivere contenuto testuale.
Esistono vari tipi di tag che permettono la formattazione.

a

Collegamento ipertestuale: Visita il sito UNIPR.

abbr

Abbreviazione: UNIPR.

b

Parola chiave: HTML è un linguaggio di markup.

br

Interruzione di linea: Prima della linea
Questa è la seconda linea.

code

Esempio di codice: <!DOCTYPE html>

data

Data in formato leggibile: 8 ottobre 2024.

dfn

Termine definito: HTML è il linguaggio di markup usato per creare pagine web.

i

Testo in corsivo: Questo è un esempio di testo in corsivo.

Preformatted Text

            Pre formatted text
            Mantiene spazi e
            interruzioni di riga.

u

Testo sottolineato: Questo è un esempio di testo è sottolineato.
